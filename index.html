Über diese App 🌸

Was diese App macht

Eine To-Do-App, die für dich da ist – ohne Druck, ohne Stress. Du siehst nur das, was du brauchst.

Schlechter Tag?

Schalte den "Schlechter Tag"-Modus ein – die App zeigt nur 3 Aufgaben und erinnert dich daran, dass eine einzige Aufgabe schon zählt.

Aufgaben hinzufügen

Tippe unten etwas ein und drücke +. Tippe auf eine Aufgabe, um sie abzuhaken.

Was diese App niemals tut

❌ Keine Streaks oder "Du hast X Tage verpasst"
❌ Kein Druck, alles auf einmal zu erledigen
❌ Keine Schuldgefühle
✅ Nur sanfte, liebevolle Begleitung

Technik dahinter

Reines HTML, CSS & JavaScript – ohne schwere Frameworks.
Google-Login über Firebase Authentication.
Installierbar als PWA mit Service Worker (Auto-Update).
Aufgaben werden lokal auf dem Gerät gespeichert.
Gehostet auf GitHub Pages, automatisch per GitHub Actions.
